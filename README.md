# Daring Pravda Theme

This directory is structured to be pushed directly to a standalone GitHub repository for Micro.blog.

## Repo Root Layout

- `theme.toml`
- `layouts/`
- `static/`

## Publish

1. Create a new GitHub repository.
2. Copy the contents of this directory to the root of that repository.
3. Push to GitHub.
4. In Micro.blog, create a new custom theme and import it from that GitHub repository.
